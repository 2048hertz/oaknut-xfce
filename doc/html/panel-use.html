Using the XFce Panel

When you start the panel you will see a bar apearing at the bottom of your screen, looking like this:

Figure 1. Default XFce panel

The figure shows the default panel configuration with one open subpanel and the mouse hovering over the setup button, making it show a tooltip.

The two strips on either side of the panel are move handles which you can use to drag the panel anywhere on the screen. Right clicking (or Shift-click) shows a popup menu — go on, try it ;-)

The area between the move handles contains panel items. There are several types of items, for instance program launchers, a clock and a desktop switcher.

Most items perform an action when clicked. Right-clicking will open a popup menu, from which you can open a dialog to change item properties, remove an item or add a new item to the panel. Look at the section called “Customization” for an explanation of all available panel items

Panel items can have a subpanel associated with it, which can be opened by clicking on the button next to the item. Use the right mouse button to open a dialog to change the properties of the items on the subpanel.

Panel preferences can be changed from the XFce settings manager, which can be started either from the 'Setup' button on the panel (see Figure 1) or from the right-click menu on the move handles.
