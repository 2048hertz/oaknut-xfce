Using the XFce Panel

Panel Layout

When you start the panel you will see a bar apearing at the bottom of your screen, looking like this:

Figure 1. Default XFce panel

The figure shows the default panel configuration with one open subpanel and the mouse hovering over the setup button, making it show a tooltip.

The two strips on either side of the panel are move handles which you can use to drag the panel anywhere on the screen. Right clicking (or Shift-click) shows a popup menu. Go on, try it ;-)

The area between the move handles contains panel items. There are several types of items, for instance program launchers, a clock and a pager.

Panel preferences can be changed from the XFce settings manager, which can be started either from the 'Setup' button on the panel or from the right-click menu on the move handles. See the section called “Panel Preferences”.

 

Panel Items

Most items perform an action when clicked. Right-clicking will open a popup menu, from which you can open a dialog to change item properties, remove an item or add a new item to the panel.

You can add panel items from the right-click menu on another item or the move handles. To remove panel items use the right-click menu or the 'Remove' button in the item properties dialog.

The properties dialog can be opened from the right-click menu on the item. Each type of panel item has its own configuration dialog. You can also change to another item type in this dialog, e.g. turn a launcher icon into a clock or vice versa.

Subpanels

Panel items can have a subpanel associated with it, which can be opened by clicking on the button next to the item.

Use the 'Add laucher' item at the top of a subpanel to add an item to a subpanel or drag an executable to this item from a file manager. To remove items from a subpanel use the 'Remove' button in the item properties dialog.

The properties dialog for a subpanel item is opened immediately when clicking the right mouse button. The dialog itself is very similar to the dialog for Launcher type panel items.
