Introduction

The XFce panel is part of the XFce Desktop Environment and features program lauchers, popup menus, a desktop switcher and more. The actual command to run is xfce4-panel. Many aspects of the panel can be configured through graphical dialogs.

XFce is a collection of programs that together provide a full-featured desktop environment. It aims to be fast and lightweight, while still being visually appealing and easy to use. The following programs are currently part of XFce:

Window Manager (xfwm4)

Handles the placement of windows on the screen and provides decorative borders.

Panel (xfce4-panel)

The program described in this manual.

Desktop Manager (xfdesktop)

Sets a background color or image and provides a menu when you click on the desktop background.

File Manager (xffm)

Fast file manager.

Utilities

Several smaller programs and scripts, for instance a taskbar (xftaskbar4) and a run dialog (xfrun4).

For more information visit the XFce website.
